Getting Started#

Setting up a Hugo blog with PaperMod is straightforward:

  1. Install Hugo
  2. Create a new site: hugo new site my-blog
  3. Add PaperMod theme as a submodule
  4. Configure your hugo.toml file
  5. Create content and start blogging!
